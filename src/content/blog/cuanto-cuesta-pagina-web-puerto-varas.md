## Factores que determinan el precio de una pagina web

### Tipo de sitio web

El costo cambia segun la complejidad del proyecto:

- **Landing page o sitio one-page**: desde $150.000. Ideal para negocios que necesitan presencia basica con informacion de contacto y servicios.
- **Sitio web corporativo multi-pagina**: desde $350.000 hasta $550.000. Incluye inicio, servicios, blog, contacto y secciones personalizadas.
- **Tienda online o e-commerce**: desde $650.000 hasta $1.200.000. Agrega carrito de compras, catalogo de productos y pasarela de pago como Webpay.
